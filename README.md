# Kit Escolar 2026 — Controle de Entregas

Sistema de gestão operacional para acompanhamento da distribuição de kits escolares às 189 unidades da rede municipal de Cuiabá-MT.

**Acesso:** https://controle-kit-escolar-cuiaba.github.io/KIT_2026/

## Perfis de acesso
- **Administrador:** gerencia status, estoque e grupos
- **Leitor:** acompanha o progresso em tempo real

## Dados operacionais
- 189 unidades escolares
- 59.673 kits no total
- 22 grupos de entrega (16/04 → 19/05/2026)
- Limite de 2.500 kits por dia

## Tecnologia
Sistema HTML único auto-contido — sem servidor, sem banco de dados, funciona offline após o primeiro acesso. Hospedado via GitHub Pages.
